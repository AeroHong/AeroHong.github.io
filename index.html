영상 프레임 단위 탐색 + 터치·클릭 위치 추적 (자유낙하 / 수평투사 각각)

현장에서 직접 부딪히며 쌓은 설계 철학입니다.

🏫 현장 중심 설계

고교학점제 현실에 맞춰 선택과목·분반·보강·고사 업무까지 현장의 실제 흐름을 그대로 반영한 시스템을 설계합니다. "있어 보이는 기능"보다 "실제로 쓰이는 기능"이 우선입니다.

🎯 사용자 중심 설계

교사와 학생 모두가 별도 교육 없이 바로 쓸 수 있도록 불필요한 UI·UX를 철저히 배제합니다. 꼭 필요한 기능만 남긴 간결한 인터페이스가 실사용률을 높입니다.

⚡ 빠른 피드백 반영

사용자 피드백을 현장에서 직접 수집하고 빠르게 수정·개선합니다. 개발자와 사용자 사이의 거리가 없기 때문에 현장에서 문제가 발생하면 당일 패치도 가능합니다.

🌐 범용성 설계

단위 학교만을 위한 시스템에 머물지 않고, 모든 학교가 안전하고 쉽게 사용할 수 있는 멀티테넌트 SaaS 구조로 설계합니다. 이메일 도메인 하나로 어느 학교든 즉시 사용 가능합니다.
